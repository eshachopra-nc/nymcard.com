# NymCard References Analysis

This document captures visual and interaction references that influence the future direction of nymcard.com.

The purpose of this file is not to copy competitors directly.

The purpose is to:
- identify strong interaction patterns
- understand modern infrastructure storytelling
- study premium UI composition
- extract motion principles
- identify layout systems
- define visual language patterns for NymCard

Primary reference weighting:
- Stripe → layout, gradients, infrastructure storytelling, UI-led sections
- Linear → motion continuity, AI-native workflows, floating operational UI

---

# Stripe References

---

## stripe-home-hero

### What it is
Hero section with left-aligned F-pattern typography and a massive restrained gradient field flowing diagonally across the right side of the page. Includes minimal CTAs, global proof (“Global GDP running on Stripe”), and trusted enterprise logos.

### Why we like it
- Instantly communicates “global financial infrastructure” rather than SaaS startup
- The gradient behaves like ambient lighting, not decoration
- Strong asymmetric composition creates movement without chaos
- UI-free hero still feels alive because of scale, gradients, and composition
- The left-heavy editorial layout feels premium and institutional
- Excellent whitespace balance without feeling empty
- Strong benchmark for NymCard’s homepage hero direction
